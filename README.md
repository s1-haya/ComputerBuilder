# Computer-Builder

# projectの説明
PCの部品(CPU、GPU、RAM、STORAGE)を選択し、PCを構築します。
PCの性能はゲーミング用と作業用のスコアによって確認できます。
数ある部品組み合わせて最適なPCを構築してみてください！
